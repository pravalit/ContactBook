# 📒 Contact Book App

A simple full-stack **Contact Management Web App** built using HTML, CSS, JavaScript, and Node.js.

---

## 🚀 Live Demo

- 🌐 Frontend (GitHub Pages):  
  https://pravalit.github.io/ContactBook/

- ⚙️ Backend API (Render):  
  https://contactbook-47lo.onrender.com/contacts

---

## ✨ Features

- ➕ Add new contacts  
- 🗑️ Delete contacts  
- 🔍 Search contacts by name  
- 🏷️ Filter by category (Family / Friends / Work)  
- 🌐 Fully deployed (Frontend + Backend)

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Node.js, Express  
- **Hosting:**  
  - GitHub Pages (Frontend)  
  - Render (Backend)

---
## ⚙️ How it Works

- Frontend sends requests using `fetch()`  
- Backend handles:
  - GET → fetch contacts  
  - POST → add contact  
  - DELETE → remove contact  
- Data is stored in a JSON file

---

## ⚠️ Note

- Backend is hosted on Render (free plan)  
- It may **sleep after inactivity**  
- First request may take **20–30 seconds**
